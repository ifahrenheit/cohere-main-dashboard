# Cohere Main Dashboard

Employee management and timesheet tracking system.

## Features
- Employee Management
- Timesheet Tracking
- OT/CWS/FTS/RDWork Approvals
- Quiz Module
- Google Sheets Integration

## Setup
1. Clone repository
2. Copy `db_connection.example.php` to `db_connection.php`
3. Configure database credentials
4. Import database schema
